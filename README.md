# NayePankh Foundation Website

## Overview

NayePankh Foundation is a responsive NGO website developed to support the organization's mission of empowering underprivileged communities through education, health awareness, food distribution, and social welfare initiatives.

This project was created as part of the Front-End Development Internship Task.

---

## Features

### Responsive Design

* Mobile-friendly layout
* Tablet and desktop support
* Optimized user experience across different screen sizes

### Modern User Interface

* Clean and professional NGO-themed design
* Smooth scrolling navigation
* Interactive sections and cards
* Dark mode support

### Hero Section

* Dynamic background image slider
* Mission-focused content
* Call-to-action button

### About Section

* Organization overview
* Mission and vision highlights

### Programs Section

* Education Support
* Health Awareness
* Food Distribution
* Community Welfare

### Volunteer Section

* Volunteer registration form
* Community engagement information

### Donate Section

* Support and contribution awareness section
* NGO impact highlights

### Contact Section

* Contact information
* Contact form for inquiries
